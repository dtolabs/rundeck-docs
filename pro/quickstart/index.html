How can I try out Rundeck Pro?

Note: this setup is meant for evaluating Rundeck on your local workstation. For a guide to setting up Rundeck Pro in production see Installation.

  1. Fill out the download form to access the download page.
  2. Click on âž¤ Cluster and click on the the file ending in .jar to start the download.

  3. Once the download is finished verify that the file's checksum matches the expected checksum:

    shasum -a 1 ~/Downloads/rundeckpro-launcher-cluster-3.0.x-SNAPSHOT.jar

  4. Run the .jar file:

    java -XX:MaxPermSize=256m -Xmx1024m -jar ~/Downloads/rundeckpro-launcher-cluster-3.0.x-SNAPSHOT.jar

  5. Once you see something similar to following log output, you know the server is ready:

    2018-04-23 16:18:58.889:INFO:oejs.ServerConnector:main: Started ServerConnector@2dcf960d{HTTP/1.1}{0.0.0.0:4440}
  6. Navigate to http://localhost:4440/ in a browser

  7. Log in with the username admin and password admin

Rundeck Pro is now running on your workstation for you to evaluate!
