Version 3.0.6

Release 3.0.6

Date: 2018-09-17

Name: “jalapeño popper deeppink cutlery”

Notes

This release contains bug fixes, including some UI changes to improve the sidebar behavior, and one enhancement to allow admins to list the User summary information.
